# -Touhou Aim Lord-
这个项目是学习游戏开发的练习作品，用于学习如何实现一个极其简易的，用C编写的，基于 EasyX、网上找的不知名GDI+透明贴图库 开发的东方Project自机狙练习器（因为自机狙最好实现）。我甚至把ZUN史山代码的左右键同时按会只向左的bug复现了。作者仅是高中生，学业繁忙，故水平有限，时间也有限，本项目目前只是随便做出了一点样子，还有些问题，留档一下，之后会修。

## 项目特性
- **语言：**：完全用C编写
- **依赖：**：EasyX ，GDI+


## 按键
- **方向键**：移动
- **Z键**：发射子弹
- **Shift键**：低速模式（有低速阵法旋转特效）

## 其他：  
- **power超过60会切换为双排子弹。**
- **2UN经典控制Bug复刻**
- **BGM是将我最喜欢的一首永夜抄的BGM，用AI改编得到**
